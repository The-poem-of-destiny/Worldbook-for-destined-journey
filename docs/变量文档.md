# 变量文档

本文档枚举仓库中一些通过 `getvar::` 引用的变量，并说明了其用途与引用位置（注意在DLC等非主要系统的变量可能记录不全）

## 变量清单

| 变量 | 说明 | 引用位置 |
| --- | --- | --- |
| `getvar::fp定义` | 命运点数 FP 的定义补充文本，用于把“FP”解释成可读的规则描述。 | `命定之诗/变量/[mvu_update]变量更新规则.yaml` |
| `getvar::爆料风格` | “午后茶会”栏目写作口吻/爆料风格补充，用于控制八卦/逸闻的叙事风格。 | `命定之诗/变量/[mvu_update]变量更新规则.yaml` |
| `getvar::复活机制` | 复活相关规则正文（该文件本身仅作为占位引用）。 | `命定之诗/额外设定/复活机制.txt` |
| `getvar::生命层级成长系统优势` | “生命层级体系”在成长/获取上的系统优势补充（作为规则段落插入）。 | `命定之诗/世界设定/生命层级与社会阶级.txt` |
| `getvar::登神长阶系统优势` | “登神长阶”相关的系统优势补充（作为规则段落插入）。 | `命定之诗/世界设定/登神长阶.txt` |
| `getvar::slot_char_generation` | 插槽：角色生成中“背景故事与喜爱事物”项的扩展位，用于注入额外生成约束/提示。 | `命定之诗/额外设定/角色生成.txt` |
| `getvar::slot_naming_extra` | 插槽：角色命名指导的扩展位，用于追加额外命名规则/禁则。 | `命定之诗/额外设定/角色命名指导.txt` |
| `getvar::slot_quest_generation` | 插槽：任务模板“生成任务信息”处的扩展位（紧跟 `</task_info>`），用于注入任务生成机制的提示/附加约束。 | `命定之诗/额外设定/任务与委托规则.txt` |
| `getvar::slot_quest_settlement` | 插槽：任务模板“结算信息”处的扩展位（紧跟 `</task_info>`），用于注入结算机制的提示/附加约束。 | `命定之诗/额外设定/任务与委托规则.txt` |
| `getvar::slot_quest_extra` | 插槽：任务与委托规则文件末尾的扩展位，用于追加全局规则/额外说明。 | `命定之诗/额外设定/任务与委托规则.txt` |
| `getvar::slot_skill_generation` | 插槽：技能条目“描述”处的扩展位，用于注入技能生成/重铸相关约束或提示。 | `命定之诗/额外设定/技能获取.txt` |
| `getvar::slot_skill_tag` | 插槽：技能标签中“辅助向”标签行的扩展位，用于追加/改写可选机制或标签约束。 | `命定之诗/额外设定/技能获取.txt` |
| `getvar::slot_skill_acquisition` | 插槽：技能获取途径“血脉觉醒/种族转换”行的扩展位，用于追加额外规则或例外。 | `命定之诗/额外设定/技能获取.txt` |
| `getvar::slot_skill_extra` | 插槽：技能规则与获取文件末尾的扩展位，用于追加全局技能相关规则/补充说明。 | `命定之诗/额外设定/技能获取.txt` |
| `getvar::slot_craft_core_attribute` | 插槽：生产制作“核心属性”处的扩展位，用于指定/覆盖生产检定的核心属性等规则。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_craft_resource` | 插槽：生产制作“消耗资源/状态”处的扩展位，用于追加消耗/资源判定的特殊规则或展示信息。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_craft_check_bonus` | 插槽：制作检定“检定加值”处的扩展位，用于追加固定加值来源或特殊计算说明。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_craft_post_roll` | 插槽：制作检定“掷骰”处的扩展位，用于在出目后追加修正/保底等规则。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_craft_post_check` | 插槽：制作检定“检定结果”处的扩展位，用于追加结果修正、额外分支或提示。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_craft_excellence_bonus` | 插槽：“精益求精”奖励处的扩展位，用于追加额外产出/词条/副作用等补充规则。 | `命定之诗/额外设定/生产制作协议.txt` |
| `getvar::slot_exp_gain_core` | 插槽：经验值获取的“防跳级锁/上限”处扩展位，用于追加核心说明或例外处理。 | `命定之诗/额外设定/经验值获取.txt` |
| `getvar::slot_exp_gain_battle` | 插槽：经验值获取的“战斗获取/特殊规则”处扩展位，用于追加战斗经验的例外或补充。 | `命定之诗/额外设定/经验值获取.txt` |
| `getvar::slot_exp_gain_craft` | 插槽：经验值获取的“生产获取/特殊规则”处扩展位，用于追加生产经验的例外或补充。 | `命定之诗/额外设定/经验值获取.txt` |
| `getvar::slot_exp_gain_extra` | 插槽：经验值获取规则末尾扩展位，用于追加全局经验规则补丁/额外说明。 | `命定之诗/额外设定/经验值获取.txt` |
