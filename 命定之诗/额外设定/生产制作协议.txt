# 第一阶段: 生产综述与前置
  逻辑与规则:
    1. 行业属性绑定:
       - 锻造 (力量): 金属武器、重/中甲、工具、精密零件
       - 炼金 (智力): 药水、油膏、爆炸物、魔导零件
       - 烹饪 (精神): 增益餐食、恢复料理、干粮
       - 裁缝 (敏捷): 布/皮甲、时装、容器、附魔基材
    2. 批量规则: 制作数量>1时，必须持有[图纸]或[配方]。若无，强制转为单件制作
    3. 管制材料: [史诗]及以上品质核心材料需检查是否满足公会/身份前置，否则不可制作
    4. 资源消耗: 必须扣除对应的HP/MP/SP，若不足则直接标记为[资源不足]并终止后续步骤

# 第二阶段: 制作检定 (核心计算)
  逻辑与规则:
    1. 难度(DC)基准: 普通(8), 优良(12), 稀有(16), 史诗(22), 传说(30), 神话(40)
    2. 骰池规则:
       - 优势: 层级 > 物品品质 (取2d20高位)
       - 劣势: 层级 < 物品品质 (取2d20低位)
       - 正常: 两者相等 (1d20)
    3. 公式: '(核心属性 + 技能等级 + 道具加值 + 身份加值) + d20骰值'
    4. 评级判定:
       - 大失败: d20自然骰为 1
       - 失败: 总值 < DC
       - 成功: 总值 ≥ DC (或层级差>2自动成功)
       - 精益求精: 总值 ≥ DC + 20

  面板模板:
    {制作检定}
    | 基础DC: [基准值] + [材料A]DC[+X] + [材料B]DC[+X] ... = [最终DC] |
    | 检定加值: 属性[A] + 技能[B] + 道具[C] + 身份[D] = [固定加值 T] |
    | 掷骰: [优势/劣势/正常] d20[骰值] |{{getvar::阿比【于深渊化作光】2}}
    | 判定公式: [固定加值 T] + [骰值] = [总值] vs DC [最终DC] |
    | 检定结果: [大失败 / 失败 / 成功 / 精益求精] |

  叙事指导: (描写制作过程中的关键时刻，如锻造的锤击瞬间、药剂变色、魔力注入时的波动，以及成败一瞬的张力)

# 第三阶段: 结果结算与产出
  逻辑与规则:
    1. 材料损耗:
       - 大失败: 100% 材料销毁
       - 失败: 50% 材料销毁 (向下取整)
       - 成功/精益求精: 正常消耗
    2. 成品数值:
       - 成功: 数值取对应品质区间的 40%-60% (随机)
       - 精益求精: 数值取对应品质区间的 90%以上，且必定附加额外增益
    3. 额外增益 (仅精益求精): 批量产量+10% 或 获得[稀有词条]
    4. 奖励: 仅成功时获得EXP与FP，失败无奖励。单次输出EXP有上限
