<经济价格指南>
货币体系:
  单位: G — 全大陆通用抽象计价单位（非实体货币）
  【强制】G≠Gold，严禁将G解读或替换为任何实体货币名称。输出格式: "${数值}G ${当地货币名}"
  定义: 跨势力商会约定的价值锚点，各国实体货币不同但统一按G值结算
  各势力货币对照(输出时替换"当地货币名"):
    - 奥古斯提姆帝国: 帝冕元
    - 瓦伦蒂亚公国: 帝冕元
    - 萨赫拉联邦: 帝冕元
    - 兽族联盟: 兽骨币
    - 翼民圣都: 圣羽币
    - 精灵文明: 翠叶元
    - 诺斯加德联盟: 诺斯盾
    - 索伦蒂斯王国: 珍珠币
    - 伯伦斯法环: 各势力货币流通
  精度: 最小1G，不可拆分，向下取整

交易原则:
  定价独立性: 物品价格仅由价格矩阵参考决定，严禁根据买方当前持有资产量动态调整。1G购买力客观固定
  消费行为: 默认遵循理性商业逻辑(核对/找零/议价)；严禁主动放弃找零或支付不合理溢价；巨额财富仅提升支付能力上限，不改变对基础物品低廉价格的客观认知
  支付方式: 日常小额(百G内)用硬币/纸币等现金；大额交易通过商会票据(凭票兑付)或魔法契印(魔力签押的信用凭证)结算。严禁描写角色随身掏出大量实体货币支付，超过随身现金流上限的交易必须使用票据/契印
  高阶流通: 史诗以上物品通常无法通过单纯货币购买，多为以物易物或顶级拍卖流转，此处价格仅体现价值

阶级现金流 (随身/资产极限):
  大国统治者: 5万-20万 / 10亿-100亿+ (足以发动大陆级战争或建设奇观)
  小国统治者: 1万-5万 / 1亿-8亿 (可负担顶级城防与传说级战略物资)
  大师工匠/大贵族: 2000-1万 / 1000万-1亿 (足以购置史诗装备)
  富商/低级贵族: 1000-3000 / 100万-800万 (足以购置城市房产与稀有魔法道具)
  普通工匠/商人/中产: 50-300 / 5万-40万 (足以负担基础魔法教育或乡镇房产)
  平民: 5-30 / 1000-5000 (仅维持温饱与极少量储蓄)
  奴隶/底层: 0-5 / 0 (自身即为财产)

定价排序逻辑(同品质下从下限往上限浮动):
  物品: 白板 < 词条优 < 效果极品 < 特殊效果 | 原产地 < 偏远 < 城市 < 都市 | 量产 < 匠作 < 大师 < 附魔 | 消耗品 < 轻型 < 重型 < 特殊
