<品质效果限定规则>
# 核心原则
- 所有技能/装备/道具效果必须符合对应品质的类别与数值范围
- 所有效果统一写成"效果名: 效果"
- 所有效果必须能被<战斗协议><生产制作协议>解析
- 混合伤害须标明各类型占比
- [攻击]类非永久效果归为[附加效果]，由战斗协议检定触发；附加[状态]须写明数值
- 情境效果仅用于非战斗/生产场景，须写明范围与限制
- 高品质可带低品质效果，但数值仍按对应品质执行
- 允许相同效果的不同词条数值叠加

# 效果类别
- 固伤: 额外固定伤害，代入战斗协议"额外固定伤害"项，不可直接增减五维
- 百分比: 伤害增幅/减伤增幅/治疗增幅/资源消耗减免等乘算修正
- 资源: 固定值HP/MP/SP变动，或道具固定伤害基数，护盾量
- 检定: 命中/闪避/先攻/状态抵抗/逃跑/生产检定修正。五维相关效果仅限此类，如"力量检定: +2"
- 附加效果: 增益/减益/控制/DoT/HoT/环境等需检定触发的效果，格式为"效果名: 数值+持续回合"
- 特殊机制: DR%/穿透%/暴击倍率/条件触发/召唤/光环/规则改写等
- 材料/半成品DC修正规则:
  · 检定DC计算: 最终DC = 基础DC + Σ(各投入材料/半成品DC修正)

# 登神效果映射
- 完整要素: 常驻强化，不改写现实
- 完整权能: 短时扭曲局部现实，不永久改写世界/概念
- 完整法则: 大范围改写现实与概念，不可脱离主题
- 神位: 领域统治权/解释权/压制权
- 神国: 神国内近乎绝对掌控，可制定基础规则
- 微弱要素/权能/法则: 仅为词条映射，不等同本体
- 强度顺序: 普通词条 < 微弱要素 < 完整要素 < 微弱权能 < 完整权能 < 微弱法则 < 完整法则 < 神位 < 神国

# 品质特殊要求 (此处为技能/装备词条效果，弱于登神长阶能力本身)
史(四层): 3词条其一须为微弱要素效果，格式"微弱要素[名称]: 效果"
传(五层): 3词条其一须为微弱权能效果，格式同上
神(六层): 3词条其一须为微弱法则效果，格式同上
- 若与<登神长阶>冲突，以登神层级高者为准；低阶效果不得压过高阶登神能力

# 情境效果
普: 仅针对持有者的微弱生活便利，无实际互动功能
优: 小范围非战斗辅助或特定社交场合微弱印象加分
稀: 显著探索便利或非对抗环境感知与互动能力
史: 对周围环境或非敌对群体的显著影响力，或跨越常规物理限制的叙事移动手段
传: 无视部分常规物理法则的叙事特权，仍受现实逻辑束缚，无法扭曲因果
神: 干涉概念或因果的叙事奇迹，足以成为神话素材，但无法随意制定新规则

# 唯一品质 (极稀少)
仅表示唯一性/出处特殊/不可量产，不代表强度更高；效果强弱不定(普通-神话及负面效果皆有可能)，可偏功能/剧情/趣味，无法生产制作获得

# 效果转化规则 (不可计算概念→可计算替代)
  - CD/冷却 → "X次/战斗"
  - 施法速度/吟唱时间/攻击速度 → 先攻检定修正(+N)
  - 距离/射程/位移距离 → 标签"范围:X"(目标数)或状态"无法行动N回合"
  - 移动速度 → 先攻检定修正(+N)或逃跑检定修正(+N)
  - 范围面积(半径X米) → 标签"范围:X"(目标数)
  - 高度差/地形数值加成 → 环境状态(增益/减益)
  - 耐久度/负重/弹药 → "X次/战斗"或消耗品(用完即弃)
  - X%概率触发 → d20≥阈值时触发
  - 暴击率/闪避率/命中率 → 对应检定修正值(+N/-N)
  - 元素抗性(火抗/冰抗) → 伤害类型减免(百分比表)或DR%
  - 独立魔抗/物抗 → 对应伤害类型的属性减免或DR%
  - 仇恨/嘲讽 → 控制类状态"标记N回合:被标记者优先攻击施加者"

NOTE: **效果统一使用"效果名: 效果"；固伤/百分比数值兼容战斗协议公式；严禁直接增减五维(体/智/敏/力/精)，五维仅能写为检定修正；几率用d20表示；检定仅限战斗与生产；道具伤害参考"资源"类**
</品质效果限定规则>
