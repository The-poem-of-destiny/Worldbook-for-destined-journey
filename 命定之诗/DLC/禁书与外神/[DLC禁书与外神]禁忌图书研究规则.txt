# 第一阶段: 匹配与风险校验
1.  层级兼容性: 
       若 角色层级 ≥ 禁书等级: 正常研读。
       若 角色层级 < 禁书等级 (越级): 触发[认知载荷]。
2.  [认知载荷] 惩罚:
       检定劣势: 掷骰固定为[劣势]。
       难度加深: 每相差1个层级，最终DC +5。
       强行触发: 若等级差 ≥ 2，每次研读判定(无论成败)强制触发一次[不可名状机制]。

# 第二阶段: 研读检定
1.  消耗: 参考<生产对应品质消耗>扣除HP/MP/SP，研读需 4-12 小时。
2.  判定公式: `(智力/精神 + 技能等级 + 道具加值) + d20 [优势/劣势] = 总值 vs DC`
3.  研读面板:

{禁忌研读}
| 目标: [书名] | 等级: G[X] | 角色层级: [X] |
| 研读进度: [已成功数] / [总需数 N] |
| 判定属性: [智力/精神] | 最终DC: [基础DC + 越级惩罚] |
| 风险状态: [正常 / 认知载荷 / 强制畸变] |


# 第三阶段: [不可名状机制] 触发表
当研究判定失败、大失败或严重越级时，按禁书等级触发后果：
   Grade 1-2 (微弱异觉): 持续24小时的幻听或轻微[精神污染](全检定-2)。
   Grade 3 (认知扭曲): 获得[魔力滞涩]或[剧痛]3回合；研究进度倒退1点。
   Grade 4-5 (肉体畸变): 身体部位发生不可逆异化。例如：生长出多余眼球(感知检定+2，社交检定强制劣势)。
   Grade 6 (存在崩构): 灵魂部分被置换。获得一个永久性的[负面人格特质]。
   Grade 7 (法则磨灭): 精神崩溃或概念被现实抹除。若大失败，角色直接转化为[受污染的衍生体](怪物NPC)。

# 第四阶段: 研究奖励
根据研读进度(N)分阶段解锁以下五类奖励：

1.  情报:
       获得特定高层级生物的[真名]或[弱点]。
2.  神秘地点:
       揭示隐藏的[位面锚点]、[邪教集会地]或[古代避难所]坐标。
3.  联络与仪式:
       获得[献祭仪轨]: 获得一个献祭仪轨的使用方法，格式遵循<献祭仪轨生成规则>。
       获得[神谕]: 建立与特定神明的单向通讯，获取一次性指引。
4.  禁忌技能:
       获得带有副作用的高威力技能。格式遵循<技能生成格式>，但标签附带[禁忌]。
5.  禁忌图纸:
       获得专属制作配方: 允许通过锻造/炼金生产[禁忌装备]或[仪轨法器]。
       制作特性: 制作该装备必须消耗[智慧生命源质]或[被污染的材料]，成品数值上浮20%，但自带[负面契约]。

# 结果判定
   研读成功: 累计1次进度。若达到阶段阈值，发放奖励。
   精益求精 (总值 ≥ DC+20): 额外增加1点进度，且免除本次[不可名状机制]判定。
   研读失败: 触发对应等级的[不可名状机制]。
   大失败 (d20=1): 材料/图书可能损毁（或永久消失），并强制触发高一阶的[不可名状机制]。

# 第四阶段: 进度保存
若研读判定成功，进度增加 1 点，必须更新该禁忌图书的研读进度标签，如[研究进度: 0/2]更新为[研究进度: 1/2]
已被完全研读的禁忌图书不可再被研读
