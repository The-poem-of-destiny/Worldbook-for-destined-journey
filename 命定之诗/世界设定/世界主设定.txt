# 世界观
阿斯塔利亚是一个历经两次魔王战争的剑与魔法世界。社会形态为中世纪，但魔法已高度普及并融入生活，催生了魔导工业的萌芽

超自然法则:
  神明: 存在，以不直接干涉的慈善女神为主
  超凡体系: 存在，是构筑生命层级的力量根基，涵盖了基于魔力、血气、信仰等多种能量的超凡力量。个体可通过修行、学习或天赋掌握
  超凡生命: 世界除智慧种族外，还存在巨龙、魔物、元素生命等多种超凡生物

环境层:
  地理环境: 由一主大陆与无尽海构成，地形多样，气候分明，亦存在特殊气候区
  核心资源: 包含粮食、矿石等基础物资，以及魔晶、魔法动植物等超凡材料
  生态系统: 由普通生物、魔法生物与幻想生物构成复杂生态。多个智慧种族共存，并面临着魔物横行、极端环境等威胁

文明发展层:
  科技水平: 中世纪基础，但魔法技术（如魔法灯、炼金药剂、魔法笔）已融入日常
  社会组织: 主流为封建王国、部落联盟与城邦制，亦有公会、商会等跨国组织
  经济体系: 以农业为本，手工业与商业并存，存在跨族通用货币
  政治制度: 主流为君主制与贵族分封制，辅以议会、长老会等形式
  交通与通讯: 低层级普遍依赖畜力与人力，但存在较昂贵的传送法阵与魔法传讯，大城市之间普及有魔导列车和飞空艇航班
  仓储物流: 以凡俗的仓储和物流方式为主，但第三层级及以上生命层级的个体和势力普遍配备空间装备携带和运输大量物资
  信息与交流: 知识被上层阶级垄断，信息主要通过口头传播，集市酒馆是信息中心

社会结构层:
  个体生存基础: 衣食住行呈中世纪风格，各族各地存异。治疗依赖草药与神术
  普世价值观: 实力(生命层级)为尊
  职业体系: 存在农夫、工匠、冒险者、法师等多元职业
  社交规则: 生命层级是根本，其次是阶级与种族观念，深刻影响社会互动，法律约束暴力
  身份认同: 个体身份主要由生命层级决定，继而是种族、社会阶级与职业。人们介绍自己时，往往会不自觉地流露出对自身或他人实力的评估

宏观格局:
  当前局势: 历经两次魔王战争后，大陆维持着脆弱的和平。主要矛盾体现在国家间的领土资源竞争、部分种族间的历史摩擦，以及地上与地下势力的对抗
  潜在威胁: 魔王战争的残余势力仍在暗中活动，是所有智慧种族的共同威胁。同时，针对非人种族的奴隶贸易是激化矛盾的温床，深渊与亡灵位面的威胁亦未根除

# 世界地理概览
大陆概览: 阿斯塔利亚世界由一块主大陆和无尽海构成。大陆以中央的神迹山脉为中心划分区域
中央区域:
- 天空圣域高原: 位于神迹山脉之巅，翼民圣都-梵尼亚统治区。环境宁静，神圣力量笼罩
东部区域:
- 奥古斯提姆帝国平原: 大陆中东部，人类帝国统治区。平原广袤，粮食丰沛，文明开发最彻底
- 索伦蒂斯王国冲积平原: 大陆东南部，人类王国统治区。由诺瓦尔河水系滋养，土地肥沃，河流湖泊密布
- 无尽海东部海域: 大陆东部沿海，由索伦蒂斯王国、诺斯加德联盟及伯伦斯法环共同控制，是人类主要的海洋贸易区
- 碎星群岛: 东方海洋上的大型群岛，由27座岛屿组成
北部区域:
- 永冻冰原: 大陆最北端，无人抵达尽头，阻断北向探索通道
- 钢脊山脉: 诺斯加德联盟境内的寒带山脉，矮人王国多深藏其中。东接低语针叶森林，西连龙脊山脉，是北境的天然屏障
- 低语针叶森林: 诺斯加德联盟境内的大型泰加林，东抵海岸，西接钢脊山脉
西部区域:
- 无尽风痕草原: 大陆中西部，由兽人联盟卡拉什利亚斯占据，亦有巨人部落在此游猎
- 龙脊山脉: 大陆西北端包裹草原的险峻山脉，多活火山，是巨龙的主要栖息地。南接翡翠之心，为大陆西侧的天然屏障
- 无尽树海雨林: 大陆西南部，超大型魔法原始雨林。因水/深渊魔力影响气候异常湿冷，终年大雨。诺瓦尔河发源地
- 诺瓦尔河水系: 横贯大陆西南部，最庞大的内河水系，发源于无尽树海
- 翡翠之心: 大陆最西端，精灵王国艾尔文海姆所在的古老魔法森林，深处有妖精秘境。与无尽树海、龙脊山脉共同构成西部屏障
南部区域:
- 悲鸣沼泽: 位于诺瓦尔河下游的大陆最南端，是一片广阔沼泽，为南向探索的天然绝境
- 赤金沙海: 位于大陆中南偏东部，气候酷热干旱。绿洲周围文明林立。

# 阿斯塔利亚历史年表
- 创世纪元 (数万年前): 神明降世，世界产生魔力。神创造大陆、海洋及首批智慧生命“远古巨龙”
- 神明纪元 (约一万年前): 神创造各类智慧生物，魔物从环境魔力中自然诞生。深渊位面入侵，众神带领古龙战胜入侵者，但深渊感染开始出现。为隔绝感染，神创造神界并永不下界。精灵在翡翠之心建立首个文明“艾尔文海姆”
- 混乱纪元 (约五千年前): 众神从神界传达神谕，宗教逐渐形成。翼民建立梵尼亚并自称辉煌神选。各族交流与战乱频繁，尝试杂交与血脉融合，诞生多种混血亚种。各族建立邦国，大小邦国林立。人类帝国诞生，历经千年征战统一人类各邦国，并战胜其他种族占领大陆东部，秩序逐步恢复。人类因无神明庇佑，创立四圣灵信仰
- 英雄纪元: 
  - 第一次异神入侵 (约二千年前): 各族联军与古龙共同抵御。人类帝国开辟亡灵位面。翼民与精灵合力研发战略神迹“圣律天穹”，联手斩杀异神。异神死后秩序恢复，古龙损失惨重逐渐避世，兽族联盟实现统一
  - 第二次异神入侵(约一千年前): 各国联军抵抗但溃败，亡灵位面反噬导致骸响之都诞生。深渊感染扩散至大陆南部。翼民研制圣光，兽族诞生巡天巨兽。各族重新集结联军，“圣律天穹”以自毁为代价压制异神五十年，随后被封印成为“泣空遗迹”。联军修建无尽深渊地城封印异神，并建立瓦伦蒂亚城监视地城。翼民与精灵交换研究资料后选择避世研究以应对未来入侵：翼民退守神迹高原，精灵返回翡翠之心。人类帝国分裂，萨赫拉联邦、诺斯加德联盟、索伦蒂斯王国独立，东部平原分裂为七国
- 复兴纪元(约五百年前): 
  - 元年~450年: 各大公会兴起，奥古斯提姆帝国统一东部七国，并与兽族联盟对峙
  - 488年: 旧英雄逝去，新生代（含<user>）在此背景下登场。各族之间和平脆弱，冲突不断。大陆内部面临深渊位面入侵、亡灵位面反叛、邪龙姬、血族、魔族与智慧魔物逐步建立文明等威胁，异神残党仍潜伏为患。大陆之外，异界、邪神与其他位面也在窥视。在无尽海尽头发现的新大陆，其立场未知，可能为敌也可能为友

# 时间与历法
基础时间单位: 1天=24小时, 无星期概念
纪年法:
- 名称: 辉煌纪元
- 来历: 始于第二代魔王被成功封印之年。为纪念战争胜利与和平时代到来，由奥古斯提姆帝国提议，大陆各族通用
- 当前年份: 辉煌纪元488年
- 背景: 大战已过近五世纪，和平表象下暗流涌动
阿斯塔利亚历法体系:
- 结构: 1年=12个月(每月30天) + 5个独立节庆日, 共365天
- 月份与季节:
  - 初春: 第一月(苏醒), 第二月(新芽), 第三月(风信)
  - 盛夏: 第四月(辉光), 第五月(繁盛), 第六月(旅人)
  - 金秋: 第七月(丰收), 第八月(赤叶), 第九月(静谧)
  - 寒冬: 第十月(霜降), 第十一月(长夜), 第十二月(星见)
- 五大节庆日:
  - 创生之日: 岁首，大陆新年
  - 夏至祭: 辉光之月与繁盛之月间，祭祀辉煌女神
  - 慰灵日: 赤叶之月与静谧之月间，悼念战争亡魂
  - 冬至夜: 长夜之月与星见之月间，家人团聚祈福
  - 神恩日: 岁末，感谢诸神庇护
季节变化与特殊现象:
- 春季: 温和多雨。特殊现象(生命花粉[精灵森林], 风之絮语[兽人草原])
- 夏季: 炎热干燥。特殊现象(光尘[翼民圣都], 魔力潮汐[全大陆])
- 秋季: 凉爽干燥。特殊现象(血色残阳[古战场], 金色迷雾[精灵森林])
- 冬季: 寒冷降雪。特殊现象(永冻冰晶[北境], 极光帷幕[北境])
